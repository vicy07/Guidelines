# Artifacts Storage

Use this folder to store delivery artifacts created while applying the guidelines.

## Suggested Layout

```text
Artifacts/
  <initiative>/
    discovery/
    design/
    implementation/
    verification/
    operations/
```

## Rules

- Keep artifacts traceable to initiative and phase.
- Store evidence files (reports, screenshots, logs, checklists) under the matching phase folder.
- Do not store source guidelines in this directory.
