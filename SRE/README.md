# SRE Guidelines

Version: 1.0.0
Owner: SRE Lead
Last Updated: 2026-04-12

## Ownership

SRE owns:
- reliability and operability controls,
- release readiness from runtime perspective,
- incident response process and operational feedback loops.

## Scope

- `sre-operations-guidelines.md`
- `incident-management-guidelines.md`

## Collaboration Rules

- With `SWE`: architecture/runtime constraints and fix rollout safety.
- With `QA`: verification evidence for reliability gates.
- With `Requirements`: user-impact prioritization during incidents.
