# Delivery Playbooks

Version: 1.0.0
Owner: Repository Maintainer
Last Updated: 2026-04-19

## Purpose

Store executable, role-aware playbooks that can be run by agents or humans.

## Playbook Contract

Each playbook should include:
- Trigger condition
- Inputs (schema references)
- Execution steps
- Required outputs
- Exit criteria
- Escalation rules
