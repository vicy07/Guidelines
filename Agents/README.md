## Purpose

This folder defines role profiles for the agent roles used by this repository.

## Scope Boundary

- `Agents/` describes who each role is, what it owns, what it consumes, and what it must produce.
- `Areas/` contains the normative standards each role must follow.
- `AGENTS.md` remains the execution contract for work in this repository.

## Roles

- `BA.md`
- `PO.md`
- `UX.md`
- `SWE.md`
- `QA.md`
- `SRE.md`

## Usage Rule

Read the relevant role profile first when the task depends on role accountability, handoff expectations, or ownership boundaries.
Use the linked files in `Areas/` as the normative source of truth.
