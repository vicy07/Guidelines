# QA Guidelines

Version: 1.0.0
Owner: QA Lead
Last Updated: 2026-04-12

## Ownership

QA owns:
- test strategy and test design standards,
- execution evidence and release quality gates,
- bug reproducibility and post-fix regression verification.

## Scope

- `test-strategy.md`
- `test-case-design.md`
- `test-execution-and-reporting.md`
- `bug-verification.md`
- `post-fix-testing.md`

## Collaboration Rules

- With `SWE`: failure triage and remediation verification.
- With `SRE`: production-like validation and monitoring checks.
- With `Requirements`: behavior-level acceptance criteria alignment.
